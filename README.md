# homebrew-tap

emmett-lathrop-brown の Homebrew tap。

## Install

```sh
brew install emmett-lathrop-brown/tap/jig
```

## Formulae

| formula | 説明 |
|---|---|
| `jig` | jq-compatible JSON processor（humane errors・Swift） |

> 旧 `akira-toriyama/tap` は引退。今後は `emmett-lathrop-brown/tap` を使用。
> 各 repo の `update-tap` workflow が release ごとに formula の `url`/`sha256` を自動更新（HOMEBREW_TAP_TOKEN 設定時）。
